Exercise 1

Create a button that when clicked, shows or hides a text paragraph on the page.

Exercise 2

Create a function to find the largest number in an array of numbers.

Exercise 3

Create a function that takes an array of numbers and returns a new array containing only the even numbers.

Exercise 4

Alice is planning a vacation. She has saved 1,500 USD for the trip. Her flight will cost her 450 USD, and her hotel stay is estimated to be 700 USD. She also needs to set aside 20 USD for meals per day! Write a program to calculate how much money Alice will have left after all these expenses for other activities on her 7 day trip. Also, print the total amount of her expenses.

Exercise 5

Create an application where teachers can enter a student's name and their grades in different subjects. Display each student's average grade and a pass/fail status based on the average.If they pass their name should be in green, if they fail in red.

Exercise 6

A student is required to attend at least 75% of lectures to sit for the final exam. There were 60 lectures conducted this semester, and the student attended 45 of them. Write a script to check if the student is eligible to sit for the final exam based on attendance and alert the result.

Exercise 7

Write a JavaScript program for Car dealership. The user can enter model, year, and color. Then create several car objects using a constructor and store them in an array. Finally, loop through the array and log the details of each car in a paragraph on the page.The color of the paragraph needs to be the color of the car.
